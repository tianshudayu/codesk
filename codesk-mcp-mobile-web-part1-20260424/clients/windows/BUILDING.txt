Codesk for Windows packaging
============================

Runtime files in this folder are shared by:
- the source ZIP fallback used by cloud downloads
- the Windows PyInstaller packaging pipeline

Expected production flow:
1. Build `codesk-agent.exe`
2. Build `codesk-tray.exe`
3. Build `Codesk-Setup.exe` with the payload embedded

The final public download should be `Codesk-Setup.exe`.
