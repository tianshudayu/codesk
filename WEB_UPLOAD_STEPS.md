# GitHub Web Upload Steps

1. Create an empty GitHub repository.
2. Upload all files from this Part 1 folder to the repository root and commit.
3. Open the repository root again.
4. Upload the desktop_sync folder from Part 2 and commit.

Why two parts:
- GitHub web upload limits how many files you can add in one batch.
- Part 1 and Part 2 are split to stay under that limit.
